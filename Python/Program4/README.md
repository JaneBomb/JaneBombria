Summary: Least squares regression algorithm in Python

In my computational linear algebra course, I was tasked with creating code for the least squares regression algorithm. This program takes in a file with values stored in it. It then creates separate numpy matrices and does multiplication to find the final matrix which will hold the slope and y-intercept. It will then print the equation out and show a visual of the line with matplotlib.
