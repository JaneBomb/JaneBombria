# BearEstate - Real Estate Website
--------------------------------------------------------------------------------
## Background
BearEstate is a real estate website/project made to find affordable properties and prospective roommates, all on the same site.
Using RentCastAPI, users can search for different kinds of properties based on filters, or ask for property suggestions from the AI chat bot, powered by OpenAI.
Users can also post roommate listings and message other users directly.

With the cooperation of a team of 5 and feedback from a customer group, this project was completed within 3 months.

#### Note: This project was once published on a server, but has since been disabled. This project must now be run locally.
--------------------------------------------------------------------------------
## How to Run
### 1. Download Python and the GitHub Repository

### 2. Traverse to the correct directory on the local machine  
#### Windows
``` bash
cd %USERPROFILE%\JaneBombria\Python\Django\dev_cust5-main
```
#### Mac/Linux
```bash
cd ~/JaneBombria/Python/Django/dev_cust5-main
```

### 3. Install required dependencies using pip
``` bash
pip install -r requirements.txt
```
### 4. Traverse to directory with 'manage.py' file
#### Windows/Mac/Linux
``` bash
cd myproject
```
### 5. Run the django server
``` bash
python3 manage.py runserver
```
### 6. Open link in a broswer
``` text
http://127.0.0.1:8000/
```
