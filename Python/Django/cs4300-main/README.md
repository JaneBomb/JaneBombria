# Cinemaid - Movie Booking Website
--------------------------------------------------------------------------------
## Background
Cinemaid is a mock movie theater website, where you can browse available movies and book a seat on a specific date.
Seat bookings require a login.

This project was created by myself with the aid of Claude AI, within the span of 2 weeks.

#### Note: This project was originally hosted with Render, but it has since expired. This project must now be run locally.

--------------------------------------------------------------------------------
## How to Run the Project
--------------------------------------------------------------------------------
### 1. Download Python and the GitHub Repository  

### 2. Traverse to the correct directory on the local machine  
#### Windows
``` bash
cd %USERPROFILE%\JaneBombria\Python\Django\cs4300-main
```
#### Mac/Linux
```bash
cd ~/JaneBombria/Python/Django/cs4300-main
```  

### 3. Install dependencies with pip
```bash
pip install -r requirements.txt
```

### 4. Run the django server
``` bash
python3 manage.py runserver
```

### 5. Open link in a broswer
``` text
http://127.0.0.1:8000/
```
