## Asteroids Final
-------------------------------------------------------------------------------------------------
Unity Engine (2022)  
Summary: My first Unity game based on Asteroids. Includes screen wrapping, player controls, collisions, and attacking mechanics.

For my first Game Development class in Unity C#, I created a game based on Asteroids.
In this game, you can rotate the ship and thrust in specific directions. There are also asteroid obstacles that move around the screen, and the ship can shoot these asteroids to split them into multiple, smaller versions. When an asteroid comes in contact with the ship, the game ends. There is a timer that times how long the player survives.

## GDD 3080 - Final Project
-------------------------------------------------------------------------------------------------
Unity Engine 6  
Sumamry: A time management game based in Showa era Japan created in Unity 6. Features original art assets, created by myself, and a more detailed user interface.

For my User Interface Game Design course, I created a time management game inspired by a Japanese song called "Oninoinumani" or "While the Demon is Away" by Hanyuu Maigo. Set in Showa era Japan (1926-1989), the player must pick up food objects and deliver them to customers that spawn in at specific locations. The goal is to serve as many customers as possible before the timer is up. The user interface was designed to be dynamic and to show each different order from the customers. All the art assets were hand-made by myself, along with a majority of the code. Most of the unfamiliar features I implemented were created with the aid of YouTube tutorials.
