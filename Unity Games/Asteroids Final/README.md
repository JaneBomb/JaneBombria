## Asteroids Final
-------------------------------------------------------------------------------------------------
Unity Engine (2022)  
Summary: My first Unity game based on Asteroids. Includes screen wrapping, player controls, collisions, and attacking mechanics.

For my first Game Development class in Unity C#, I created a game based on Asteroids.
In this game, you can rotate the ship and thrust in specific directions. There are also asteroid obstacles that move around the screen, and the ship can shoot these asteroids to split them into multiple, smaller versions. When an asteroid comes in contact with the ship, the game ends. There is a timer that times how long the player survives.
