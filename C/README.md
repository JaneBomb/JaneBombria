# Bombria_ITER03
-------------------------------------------------------------------------------------------------
C  
Agile Methodology  
Summary: A C program that imitates a ride share. Developed over the span of the course semester using Agile.

In Spring 2024, I took a course to learn C. This project was completed in multiple iterations accross the semester. In this class I was first taught about Agile programming, which I put into practice for the project.

In this project, I was tasked to create a program that imitated the functionality of a Ride Share. In the program, the user can request a ride by entering the amount of miles needed to travel. The program will then give a summary of the amount of miles traveled and the final fare. These requests are then stored in an array, to keep track of all requests made. An admin is also implemented, where the min and max values for the fare can be changed. The admin can also manually add new ride share requests and view all the requests made during that session.
