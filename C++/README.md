# KineticEnergy.cpp
-------------------------------------------------------------------------------------------------
C++  
Summary: I/O input practice in C++

In the C++ course I took, basics and fundamentals of coding were taught in C++. This assignment's main purpose was to experiment with I/O input in C++.
In this assignment, I was tasked in getting user input for mass and velocity. These inputs were used to calculate the kinetic energy value, which was then displayed and printed to the screen.
